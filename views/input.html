Button Group

Button group is a great alternative to drop down as it has all the option listed, therefore requires one less click. 


                <div class="button-group">
                  <input type="button" class="inline-button" value="VIC">
                  <input type="button" class="inline-button" value="NSW">
                  <input type="button" class="inline-button" value="SA">
                  <input type="button" class="inline-button" value="WA">
                </div>
