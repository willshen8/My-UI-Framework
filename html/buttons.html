Buttons are basic UI component which is commonly used for call of actions, such as submit, close, pay, etc. There are 3 main types of buttons: float, raised and flat.

Normal Buttons

These buttons can be used in most scenarios.

Floating Buttons

Persistent or floating buttons conveniently exist on the page regarding how far down the page. It is required where an action needs to be readily available.

The float button above can be typically be positioned at the bottom of the page where user can go back to the top of the page anytime.

Flat Buttons

Flat buttons are text-only buttons, and they can be used in dialogs, toolbars or inline. This style of button adds an element of simplicity. The button fill in color once pressed.

Ghost Buttons

Ghost buttons are secondary button to your Call To Action (CTA) buttons, and the objective is for the user to see your CTA button, and given the ghost button as the secondary option. The example below show the green button as the primary one, and ghost button as the secondary one as an alternative option.

Disabled Buttons

These buttons stops the user from further interactions due to incorrect workflow or error.
